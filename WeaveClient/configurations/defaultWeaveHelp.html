Keyboard shortcuts to use inside a tool:
NOTE: Control key is Command on a Macintosh)
-----------------------------------------------------------------------------------------
(control -and- alt) key
Replace the current selection (default Weave action)

(control) key
Add to the current selection

(control -and- shift) key
Subtract from the current selection

NOTE: the commands below only work for Maps at the moment.
(shift) key
Zoom in to fit a selected area

(alt) key
Pan a map or set the viewed area in a tool when zoomed in
